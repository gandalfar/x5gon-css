X5GON CSS templates

Overview

Your primary documentation should be Foundation for Sites, version 6.4.

The foundation grid is very similar to the one from Bootstrap. In this project, we're using their xy-grid. Primary technical reason for this choice is because it allows easy vertical alignment of elements and it's a good wrapper for flexbox. The page itself is using 10 columns that are centered inside 12-column grid. This is because we're text heavy and it makes page easier to read. For your work, you might need more width, so just expand to full 12-columns width.

You can use src/pages/ to quickly mock interfaces and layouts.

I would recommend adding your (S)CSS changes to a new file under scss/components (and don't forget to add it src/assets/app.scss).

If you have any questions or need help, email Jure Čuhalev.

Getting started

1. Download a copy of x5gon-css from Github: https://github.com/gandalfar/x5gon-css.

2. Run npm install -g foundation-cli

3. Run foundation watch

After you're done with your custumisations, you have a few options:

1. Run foundation build, take CSS and JS and copy it into your project.

2. Adopt gulpfile.babel.js inside your build pipeline. There's also an example of much simpler gulpfile.
